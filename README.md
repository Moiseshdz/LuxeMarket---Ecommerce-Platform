# 🛍️ LuxeMarket - Ecommerce Platform

Aplicación web de ecommerce moderna inspirada en Magento.

## 🚀 Inicio Rápido

1. Abre `index.html` en tu navegador
2. ¡Listo!

## 📦 Estructura
```
ecommerce-magento/
├── index.html
├── css/styles.css
├── js/main.js
├── images/
└── assets/
```

## ✨ Características

- 🎨 Diseño premium
- 🛒 Carrito de compras interactivo
- 📱 Responsive
- ⚡ Rendimiento optimizado

## 🔧 Personalización

Ver `css/styles.css` para variables de color y `js/main.js` para productos.
